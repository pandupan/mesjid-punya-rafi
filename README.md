# 🕌 Mesjid Punya Rafi

Website **informasi masjid** yang menyediakan informasi jadwal sholat, kegiatan, dan profil masjid.

## ✨ Fitur

- **Informasi Masjid** — Profil dan detail masjid
- **Jadwal Sholat** — Waktu sholat harian
- **Kegiatan** — Informasi acara dan kegiatan masjid

## 🛠️ Tech Stack

- **Next.js** — React framework
- **TypeScript** — Type safety
- **Tailwind CSS** — Styling

## 🚀 Cara Menjalankan

```bash
npm install
npm run dev
```
